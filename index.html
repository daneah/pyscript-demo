Behavior with PyScript-defined element

The following image is also an img tag, but it's wrapped in a responsive-image custom element. The custom element is defined in PyScript and makes the image responsive so that it will shrink to fit its container. To avoid a flash of unstyled content, content of custom elements that aren't yet defined are initially hidden via CSS.
